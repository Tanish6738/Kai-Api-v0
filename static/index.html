Powerful Features

Built for security professionals, developers, and DevOps teams who need reliable secret detection.

Multiple Scanning Modes

Single repository scans, bulk operations up to 10 repos, and convenient URL parameter shortcuts for quick analysis.

Custom Rules Engine

Define custom regex patterns tailored to your organization's specific secret formats and security requirements.

24+ Secret Types

Comprehensive detection for AWS keys, GitHub tokens, Slack credentials, Google APIs, and many more service types.

Real-time Processing

No data storage, instant results, automatic cleanup, and optimized performance for rapid security assessments.

Advanced Filtering

Exclude specific paths, filter by file types, and configure scanning rules to focus on what matters most.

Comprehensive Documentation

Interactive Swagger UI, detailed OpenAPI specification, and extensive examples for seamless integration.
